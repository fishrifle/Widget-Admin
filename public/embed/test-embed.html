Ready for Demo! All widgets below will work for your WordPress and Drupal presentation.

📋 Quick Configuration

Replace your-domain.com with your actual domain in all embed files before demo:

// In passiton-embed.js (line 11) domain: 'your-actual-domain.com' // In WordPress plugin (line 27) 'domain' => 'your-actual-domain.com' // In README examples https://your-actual-domain.com/embed/passiton-embed.js

🔧 WordPress Integration

Plugin Method:

[passiton_widget slug="demo-widget" style="modal"]

Manual Method:

<div data-passiton-widget="demo-widget" data-style="modal"></div>

📁 Drupal Integration

Token Method:

[passiton-widget:demo-widget:modal]

Block Method:

Structure → Block Layout → Add PassItOn Widget Block

🎨 Style 1: Inline Widget

Embeds directly in page content with custom dimensions

<div data-passiton-widget="demo-widget" data-style="inline" data-width="100%" data-height="400px"></div>

🎯 Style 2: Modal Widget

Shows as a button that opens a beautiful popup overlay

<div data-passiton-widget="demo-widget" data-style="modal" data-button-text="💝 Support Our Cause"></div>
Demo Tip: Click the button above to show the modal functionality!

📌 Style 3: Sidebar Widget

Fixed position sliding tab - accessible from anywhere on the page

Look for the "💝 Donate" tab on the right side of your screen!

<div data-passiton-widget="demo-widget" data-style="sidebar"></div>

🚀 Programmatic Usage

Create widgets dynamically with JavaScript

PassItOnEmbed.create('#widget-container', { slug: 'demo-widget', style: 'inline', width: '100%', height: '300px' });
